# Isabelle Alves — Portfolio v2

## Como atualizar no GitHub

1. Abre o repositório `isabelle-portfolio` no GitHub
2. Arrasta os arquivos desta pasta para substituir os anteriores
3. Clica "Commit changes"
4. O Vercel republica automaticamente em ~2 minutos

## Estrutura das imagens

As imagens ficam em `public/images/`. Para adicionar ou trocar:
- Coloca a imagem na pasta `public/images/`
- No `src/App.jsx`, no array `cases`, troca o campo `image` pelo novo nome do arquivo

## Cases com "coming soon"

O Globo e Banco VR estão marcados com `comingSoon: true` no array de cases.
Quando o conteúdo estiver pronto, remova essa linha do case correspondente.
